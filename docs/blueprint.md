# **App Name**: Harvest Haven

## Core Features:

- Property Listing Submission: Owners can submit their farmhouse or resort property for listing with details and photos.
- Admin Dashboard: Admin reviews, approves, or rejects property listings via a secure dashboard.
- Secure User Login: Secure login functionality for both property owners and admin.
- User Registration: Allow new users to register on the website and to manage their own profile.
- Property Search and Filter: Users can search and filter available properties based on various criteria (location, size, amenities).
- AI-Powered Listing Tool: An AI-powered tool provides listing recommendations to owners when writing descriptions, which boosts their overall exposure on the site. The tool decides if location data, square footage and amenities information are suitable for a higher placement in the listings.
- Property Details Page: Detailed page for each property with descriptions, photos, and booking information.

## Style Guidelines:

- Primary color: Warm, muted gold (#D4A373) evoking fields of wheat.
- Background color: Light beige (#F5F5DC), providing a neutral backdrop.
- Accent color: Rustic orange (#A0522D) for highlighting important actions and information.
- Headline font: 'Playfair', serif, for headlines with an elegant, fashionable, high-end feel; body font: 'PT Sans', sans-serif, a humanist font with a modern look and warmth.
- Use rustic, line-based icons representing farmhouse and resort amenities and features.
- Clean and spacious layout with clear section divisions to highlight properties effectively.
- Subtle transitions and loading animations for a smooth user experience.
